# دليل الإقفالات السنوية - نظام المتكامل بلس ERP v6 (نسخة تحليلية)

## نظرة عامة
موقع ويب حديث وتفاعلي يوفر شرحاً تحليلياً عميقاً لعملية الإقفال السنوي على نظام **المتكامل بلس ERP v6**. تم تصميم هذه النسخة لتكون غنية بالمعلومات النصية المستخرجة والمحللة بدقة من لقطات الشاشة والتعليمات الفنية، مما يوفر مرجعاً شاملاً وسريعاً للمستخدمين.

## المميزات الرئيسية
- **تحليل نصي عميق:** شرح مفصل لكل حقل وخيار في معالج الإقفال.
- **تصميم بدون صور:** أداء فائق السرعة وتركيز كامل على المحتوى المعلوماتي.
- **ثيم ليلي ونهاري:** تبديل سلس بين الوضع الداكن والفاتح.
- **تفاعلية عالية:** استخدام Accordions لتنظيم مراحل الإقفال الخمس.
- **متجاوب بالكامل:** يعمل بكفاءة على كافة أحجام الشاشات.

## التقنيات المستخدمة
- HTML5, CSS3, JavaScript (Vanilla)
- Bootstrap 5.3 (RTL)
- Font Awesome 6.4
- Google Fonts (Cairo)

## كيفية التشغيل
افتح ملف `index.html` في أي متصفح حديث.
